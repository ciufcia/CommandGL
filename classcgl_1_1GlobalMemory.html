Detailed Description

Thread-safe singleton for global data storage across the application.

The GlobalMemory class provides a centralized, thread-safe storage mechanism for sharing data across different parts of the application. It uses std::any for type-erased storage, allowing storage of any type of data with string keys.

This class implements the singleton pattern and is thread-safe through the use of mutexes. All operations are atomic and can be safely called from multiple threads.

Note
This class is non-copyable and non-assignable.
