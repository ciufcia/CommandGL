Detailed Description

Main framework class that manages the graphics rendering pipeline and application lifecycle.

The Framework class serves as the central coordinator for the CommandGL graphics library, managing console output, event handling, timing, drawable objects, and the rendering pipeline. It provides a high-level interface for creating console-based graphics applications.
