\doxysubsection{Detailed Description}
Manages Windows console operations including display, input, and font configuration. 

The \doxylink{classcgl_1_1Console}{Console} class provides a high-\/level interface for interacting with the Windows console, including resizing the window, changing fonts, writing character buffers, and handling input events. It automatically saves and restores the original console state.
