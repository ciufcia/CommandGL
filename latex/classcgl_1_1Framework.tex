\doxysubsection{Detailed Description}
Main framework class that manages the graphics rendering pipeline and application lifecycle. 

The \doxylink{classcgl_1_1Framework}{Framework} class serves as the central coordinator for the Command\+GL graphics library, managing console output, event handling, timing, drawable objects, and the rendering pipeline. It provides a high-\/level interface for creating console-\/based graphics applications.
