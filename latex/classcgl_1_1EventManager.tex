\doxysubsection{Detailed Description}
Manages input events from the console and provides event handling capabilities. 

The \doxylink{classcgl_1_1EventManager}{Event\+Manager} class acts as an interface between the \doxylink{classcgl_1_1Console}{Console} and application code, polling for input events, maintaining an event queue, and providing flexible event handling through type-\/specific callbacks. It supports keyboard and mouse events, as well as console resize events.
