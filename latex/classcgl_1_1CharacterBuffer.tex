\doxysubsection{Detailed Description}
A 2D buffer for storing wide characters with specified dimensions. 

\doxylink{classcgl_1_1CharacterBuffer}{Character\+Buffer} provides a container for wide characters arranged in a 2D grid, commonly used for console output operations. The buffer maintains both size information and the character data as a contiguous wide string.
