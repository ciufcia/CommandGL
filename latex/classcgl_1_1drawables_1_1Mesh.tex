\doxysubsection{Detailed Description}
Base class for drawable objects defined by vertex geometry. 

\doxylink{classcgl_1_1drawables_1_1Mesh}{Mesh} objects are defined by a collection of 2D points and UV coordinates, and use triangle rasterization with a top-\/left fill rule.
