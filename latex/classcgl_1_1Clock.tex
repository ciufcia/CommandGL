\doxysubsection{Detailed Description}
High-\/resolution timer for measuring elapsed time and frame deltas. 

The \doxylink{classcgl_1_1Clock}{Clock} class provides functionality for timing operations, measuring frame deltas, and tracking elapsed time since creation. It uses std\+::chrono\+::steady\+\_\+clock for high-\/resolution, monotonic timing.
