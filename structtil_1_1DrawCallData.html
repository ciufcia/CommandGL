Detailed Description

Complete data for a single draw call.

Contains all the information needed to render a primitive, including geometry, transformation, blending, and effects. This structure is stored in the render target's draw call pool.

Draw call data is collected during the frame and processed during the render phase, allowing for depth sorting and batching optimizations.
