Detailed Description

Cross-platform keyboard and mouse input key code definitions.

Provides a unified key code enumeration and platform-specific mapping functions for Windows (WinAPI), Linux (evdev), and macOS (HID). Enables consistent input handling across different operating systems with automatic translation between platform-specific key representations and Textil's unified key codes.
