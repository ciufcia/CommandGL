Detailed Description

Manages Windows console operations including display, input, and font configuration.

The Console class provides a high-level interface for interacting with the Windows console, including resizing the window, changing fonts, writing character buffers, and handling input events. It automatically saves and restores the original console state.

◆ getSize()

Vector2< u32 > cgl::Console::getSize ( ) const

Gets the current size of the console window in character cells.

This function queries the console screen buffer to determine the current width and height of the console window, in character cells. The result is returned as a reference to a Vector2<u32> where x is the width and y is the height.

Returns
Reference to a Vector2<u32> containing the console's width (x) and height (y).
Exceptions
std::runtime_errorif unable to get console screen buffer info.
