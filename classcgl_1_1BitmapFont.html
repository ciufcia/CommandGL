Detailed Description

A bitmap font class for loading and rendering monospaced or proportional fonts from BDF files.

Supports Unicode codepoints, per-glyph metrics, and rendering text to a texture with color and background. Handles multi-line text, baseline alignment, and correct glyph placement using font metrics.

Doesn't support escape sequences, except for newline characters.
