Detailed Description

Base class for all filterable buffer types.

Provides a common interface for buffers that can be processed by filters. This abstraction allows the filter system to work with different data types while maintaining type safety through the derived template classes.

The base class defines common operations like sizing and clearing that are needed by the filter pipeline management system.
