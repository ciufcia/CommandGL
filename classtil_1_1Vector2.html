Detailed Description

template<arithmetic T>
class til::Vector2< T >

A 2D vector class template supporting various arithmetic operations.

Template Parameters
TThe numeric type for vector components (must satisfy arithmetic concept)

Vector2 represents a point or direction in 2D space with x and y components. It supports all standard vector operations including addition, subtraction, scalar multiplication, dot product, normalization, and magnitude calculations. The class is designed to work efficiently with both integer and floating-point types.
