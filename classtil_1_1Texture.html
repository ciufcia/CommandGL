Detailed Description

2D texture class for image data storage and manipulation

Represents a 2D grid of Color pixels that can be loaded from files, created programmatically, and sampled with various filtering modes. Textures use UV coordinates (0.0-1.0 range) for sampling, allowing resolution-independent access to image data.

Memory layout is row-major: pixel(x,y) = data[y * width + x] UV coordinates map as: U=0→left edge, U=1→right edge, V=0→top edge, V=1→bottom edge
