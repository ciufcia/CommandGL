Detailed Description

Predefined filters for common graphics operations.

This namespace contains a collection of ready-to-use filters for common graphics and terminal processing tasks. These filters provide building blocks for complex visual effects and data transformations.

The filters are organized by their primary use case and input/output types, making it easy to find appropriate filters for specific processing needs.
