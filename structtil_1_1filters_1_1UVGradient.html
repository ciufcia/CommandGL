Detailed Description

Filter that creates UV-based gradient effects.

Generates color gradients based on UV coordinates, creating smooth color transitions across surfaces. The gradient direction and colors are determined by the UV mapping.

This filter is useful for:

  • Background gradients
  • Color transitions
  • UV visualization
  • Procedural coloring effects
