Detailed Description

Unified event class containing all possible input event data.

Central event representation supporting multiple input types through a type-safe classification system. Each event has a specific type marker and contains data fields relevant to different event types.

Events are lightweight and can be efficiently copied and stored in queues. The type system allows for safe event handling without dynamic casts at the application level.

Usage pattern: cpp Event event = getNextEvent(); if (event.isOfType<KeyPressEvent>()) { handleKeyPress(event.key); } else if (event.isOfType<MouseMoveEvent>()) { handleMouseMove(event.mouseDelta); }
