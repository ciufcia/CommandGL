Detailed Description

RGBA color representation with blending operations.

Represents colors using 8-bit red, green, blue, and alpha channels (0-255 range). Provides color manipulation, conversion utilities, and static blending functions for compositing operations. Default constructor creates magenta (255,0,255,255) for easy identification of uninitialized colors.
